About

ActionCube is a total conversion of Wouter van Oortmerssen´s FPS called Cube. Set in a realistic looking environment, as far as that´s possible with this engine, while gameplay stays fast and arcade. This game is all about team oriented multiplayer fun.

Readme

  • game - information about gameplay, modes, weapons, etc

Contact and Help

Please do not ask ActionCube specific stuff on the cube forums, please use one of the following possibilities instead:

  • help and discussions on the forums
  • development, testing and gaming on IRC #actioncube Quakenet
  • report bugs on the bugtracker
  • submit patches here
  • mail to the core team, team AT action.cubers.net

Before you ask for help you should read this readme and check out the FAQ on the official website. Thank you.
