Add custom player skins in this folder.

The filenames should have the following contents:

[nickname]_ffa.jpg     for non-team modes (optional)
[nickname]_cla.jpg     for cla players
[nickname]_rvsf.jpg    for rvsf players

or

[clan-tag]__ffa.jpg     for non-team modes (optional)
[clan-tag]__cla.jpg     for cla players
[clan-tag]__rvsf.jpg    for rvsf players

The double underscore marks a clan-tag.


Examples:

set of skins for player 'hobo' (nickname has to match exactly):
hobo_cla.jpg, hobo_rvsf.jpg, hobo_ffa.jpg

set of skins for players from clan 'FOO' (nickname has to contain substring 'FOO', f.e. 'FOO_bar' or 'bar_FOO'):
FOO__cla.jpg, FOO__rvsf.jpg, FOO__ffa.jpg
