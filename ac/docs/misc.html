Crosshair

You can change your crosshair temporarly using the menu Team/Player Setup ⇒ Change Crosshair. However, to change it permanently you need to

  1. open your config/autoexec.cfg
  2. search for the line loadcrosshair default.png
  3. replace default.png with the filename of one of the images in packages/misc/crosshairs
  4. save the file

To change the size of your crosshair, search for this line in the same configuration file as above.

crosshairsize 15

Simply try a different value instead of 15.

Gib tuning

If you like excessive blood and the like, you might want to tune the gibs a bit:

gibnum N

Sets the number of gibs that spawn on a gib-worthy kill. Defaults to 6 and is limited to 1000. Do not blame us if your poor box freezes due to a gib flood.

gibspeed N

Sets the velocity of the gibs. Defaults to 5 and is limited to 20.

gibttl N

Sets the lifetime in milliseconds for the gibs. Defaults to 5000 and is limited to 15000.
